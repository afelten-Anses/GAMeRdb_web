August 1st, 2018: Fast phylogenetic tool! FasTosh web version is now available.

FasTosh (i.e "FAST taxOnomy with maSH") is a phylogenetic tools developed by Pauline Barbet. FasTosh aims to fastly build a rooted tree based on k-mers using the neighbour joining clustering method.

  • To use Fastosh, type directly strains IDs in the form or send them in a txt file (one id per line).
  • As a result, you obtain a tree visualization. Then it's possible to download matrix and tree files. You can also launch a new analysis.
