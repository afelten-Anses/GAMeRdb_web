April 19th, 2018 : first version of the website (open beta)

Some pages are now in opened to the public as beta version:

  • "Genomes Workspace" pages (Clostridium,Listeria,Salmonella,Staphylococcus): In these pages, you'll find any information whe have about all genomes processed by ARTwork and saved in the GAMeRdb database
  • "Genomes Workspaces" tutorial: a quick interactive guide showing how to fully exploit functionalities included in the genomes workspaces pages. This one can be accessed on every footer of the website.
