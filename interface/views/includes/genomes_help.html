SPECIES WORKSPACE

Select strains and donwload their bioinformatics data

For a better user experience, please try the interactive tutorial


You can use various selection modes

  • Left click: simple selection
  • CTRL+left click: one per one incremental selection
  • CTRL+letf click : multiple selection
You can mix these 3 selections modes.
Please notice you can also:
  • See or download these files one per one by click on the clickable links
  • Sort the rows alphanumerically by clicking on its headers

FASTOSH TOOL: FAST PHYLOGENIES

FasTosh



  • Fastosh aims to fastly build a rooted tree based on k-mers using the neighbour joining clustering method.
  • To use Fastosh, type directly strains IDs in the form or send them in a txt file (one id per line).
  • As a result, you obtain a tree visualization. Then it's possible to download matrix and tree files. You can also launch a new analysis.
