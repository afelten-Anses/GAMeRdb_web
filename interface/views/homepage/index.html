Welcome to the GAMeRdb genomic interface database


This website is currently a opened beta. In the final version, you'll find any information about all genomes saved in the GAMeRdb database, and you'll be able to screen the database by running some tools to making fast bioinformatics analyses as Blast and phylogenies (k-mers based) The following pages are currently available:

  • "Genomes Workspace" pages (Clostridium,Listeria,Salmonella,Staphylococcus): In these pages, you'll find any information whe have about all genomes processed by ARTwork and saved in the GAMeRdb database
  • "Genomes Workspaces" tutorial: a quick interactive guide showing how to fully exploit functionalities included in the genomes workspaces pages
  • Homepage: A quick resume of the GAMeRdb database statistics (Species/CC/ST/Serovar distribution) , newsfeeds, and more.

You can access to the tutorials and some external links on each footer of the website. Please contact us for any suggestion or bug report.
