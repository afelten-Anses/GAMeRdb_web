Welcome to the GAMeRdb genomic interface database


This website is currently a 2.1 version. You can find any information about all genomes saved in the GAMeRdb database, and you can screen the database by running some tools to making fast phylogeny analysis based on k-mers approach (FasTosh workflow).

In the next 6 months, we hope to propose the list of antibiotic resistance and virulence genes found for each genome present in the database (using GENIAL workflow).

The following pages are currently available:
  • "Genomes Workspace" pages (Bacillus, Clostridium, Listeria, Salmonella, Staphylococcus): In these pages, you'll find any information whe have about all genomes processed by ARTwork and saved in the GAMeRdb database
  • "Genomes Workspaces" tutorial: a quick interactive guide showing how to fully exploit functionalities included in the genomes workspaces pages
  • Homepage: A quick resume of the GAMeRdb database statistics (Species/CC/ST/Serovar distribution) , newsfeeds, and more.

You can access to the tutorials and some external links on each footer of the website. Please contact us for any suggestion or bug report.
